# 영어 타자 연습기 
한국디지털미디어고등학교 프로그래밍 프로젝트로 제작한
프로그래밍 수업 시간에 배운 내용를 중점으로 이용하여 구현한 **영어 타자 연습기**입니다.

## 소개
영어 단어나 문장을 따라 치고 시간과 정확도를 측정할 수 있도록 만든 프로그램입니다.
프로그래밍 수업 시간에 배운 내용을 잘 활용하면서 평소에 타자 실력을 늘릴 수 있으면 좋겠다는 마음에서 제작하게 되었습니다.

메뉴에서 단어 연습, 문장 연습, 장문 연습을 선택하고 그에 맞는 영어가 출력되고 이를 따라치면 친 문자들의 정확도와 시간, 1분 당 타자 수를 알려줍니다.
그후 결과를 저장하고 메뉴의 기록에서 다시 확인할 수도 있습니다.

## 개발 환경
- OS `Window 11`
- 사용 언어 `c17`
- 사용 컴파일러 `cl`
- 사용 IDE `visual studio code`
